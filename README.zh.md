## 📌 项目介绍
MiniSnap 是一款专为 **Instax Mini 99** 设计的拍摄辅助工具。通过设备摄像头实时评估人脸和场景参数，它能即时提供智能的拍摄建议。项目的核心理念是死死锚定最关键的“面部曝光”要素，帮助用户轻松避开拍立得常见的环境过曝与欠曝问题。

## ✨ 核心功能
* **拍摄模式建议**：自动推荐最合适的拍摄模式（Normal / Indoor / Bulb）。
* **对焦距离档位建议**：根据拍摄物距离，精准提示物理镜头环应调整的位置（Macro / Standard / Landscape）。
* **曝光控制档位建议**：智能推荐准确的亮度补偿挡位（L+ / L / N / D / D-）。
* **闪光灯使用建议**：智能评估何时应该开启填充闪光灯（Fill）或关闭（Off）。
* **Mini 99 取景视角匹配**：实时预览画面等效裁剪（Zoom），模拟 Mini 99 真实的取景视野。
