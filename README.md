> An Instax Mini 99 shooting assistant tool that helps you get the perfect shot every time.

## 📌 Introduction
MiniSnap is a simplified shooting assistant designed specifically for the **Instax Mini 99**. By evaluating real-time face and scene parameters through your device's camera, it provides intelligent and immediate shooting guidance. Our goal is to anchor on the most important factor—facial exposure—to help you avoid common Instax overexposure and underexposure issues.

## ✨ Key Features
* **Shooting Mode Recommendation:** Automatically suggests the best mode (Normal / Indoor / Bulb).
* **Focus Distance Recommendation:** Tells you exactly where to set the lens ring (Macro / Standard / Landscape) based on subject distance.
* **Exposure Control (EV) Recommendation:** Recommends precise exposure compensation (L+ / L / N / D / D-) to protect highlights.
* **Flash Mode Recommendation:** Intelligently advises when to use Fill Flash or keep it Off.
* **Mini 99 Viewfinder FOV Matching:** Real-time preview zoom that simulates the actual Mini 99 viewfinder perspective.
